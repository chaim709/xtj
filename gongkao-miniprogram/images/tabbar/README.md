## 图标规格

- 格式：PNG（支持透明背景）
- 尺寸：81x81 像素（@2x: 162x162，@3x: 243x243）
- 未选中状态颜色：#999999
- 选中状态颜色：#1890FF

## 获取图标

可以从以下网站获取免费图标：

### 阿里巴巴矢量图标库（推荐）
- https://www.iconfont.cn/
- 搜索关键词：首页、日历/课表、学习/书本、我的/用户
- 下载时选择 PNG 格式，颜色设置为 #999999 和 #1890FF

### 字节跳动 IconPark
- https://iconpark.oceanengine.com/
- 支持自定义颜色和大小

### Remix Icon
- https://remixicon.com/
- 开源免费，风格统一

## 注意事项

1. 微信小程序 TabBar 图标必须是本地图片，不支持网络图片
2. 图标大小限制：40KB 以内
3. 建议使用纯色图标，避免过于复杂的图案
4. 确保选中和未选中状态的图标形状一致，仅颜色不同
