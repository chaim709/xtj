### 步骤1: 测试基础功能

1. **在微信开发者工具中点击「编译」**
2. **查看模拟器**：

### 步骤2: 检查 Console 错误

在开发者工具中：
1. 点击右侧「调试器」tab
2. 查看「Console」面板
3. **截图发给我，我来分析**

常见错误：
```
❌ Component is not found in path
❌ require is not defined  
❌ Cannot read property 'xxx' of undefined
```

### 步骤3: 检查构建 npm 状态

在开发者工具中：
1. 点击「详情」tab
2. 查看「本地设置」：
   - [ ] 是否勾选「使用 npm 模块」

如果没有勾选，请勾选后：
```
工具 → 构建 npm → 清缓存 → 编译
```

### 步骤4: 查看 miniprogram_npm 内容

确认是否正确构建：
```bash
cd /Users/chaim/.cursor/worktrees/____/vrt/gongkao-miniprogram
ls -la miniprogram_npm/
```

应该看到 `@vant` 文件夹。
