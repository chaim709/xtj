# 🧠 常识判断

常识判断考查广泛的知识面，涉及政治、法律、经济、历史、地理、科技等多个领域。

## 考情概述

| 考试类型 | 题量 | 分值 | 难度 |
|---------|------|------|------|
| 国考 | 20题 | 约10分 | ⭐⭐⭐ |
| 省考 | 15-20题 | 约10分 | ⭐⭐⭐ |

## 知识模块

### 一、政治理论

| 考点 | 内容 |
|------|------|
| 马克思主义 | 唯物辩证法、历史唯物主义 |
| 毛泽东思想 | 新民主主义革命理论 |
| 中国特色社会主义 | 习近平新时代思想 |
| 党史党建 | 重要会议、历史事件 |

#### 高频考点

**习近平新时代中国特色社会主义思想：**
- "五位一体"总体布局
- "四个全面"战略布局
- 新发展理念（创新、协调、绿色、开放、共享）
- 高质量发展

**重要会议：**

| 会议 | 主要内容 |
|------|---------|
| 十八大 | 全面建成小康社会目标 |
| 十九大 | 新时代中国特色社会主义思想 |
| 二十大 | 中国式现代化 |

### 二、法律常识

| 法律 | 重点内容 |
|------|---------|
| 宪法 | 公民基本权利义务、国家机构 |
| 民法典 | 物权、合同、婚姻家庭、继承 |
| 刑法 | 犯罪构成、刑事责任、常见罪名 |
| 行政法 | 行政处罚、行政许可、行政诉讼 |

#### 高频考点

**宪法基础：**
- 国家根本制度：社会主义制度
- 根本政治制度：人民代表大会制度
- 基本政治制度：多党合作、民族区域自治、基层群众自治

**刑法常考：**
- 刑事责任年龄：16周岁完全责任，12-16周岁相对责任
- 正当防卫与紧急避险的区别
- 故意与过失的区分

**民法典要点：**
- 民事行为能力年龄：18周岁完全，8-18周岁限制
- 诉讼时效：一般3年，最长20年
- 继承顺序：第一顺序（配偶、子女、父母）

### 三、经济常识

| 考点 | 内容 |
|------|------|
| 微观经济 | 供需理论、市场结构、消费者行为 |
| 宏观经济 | GDP、通货膨胀、财政货币政策 |
| 国际贸易 | WTO、自贸区、一带一路 |

#### 高频考点

**GDP与GNP：**
- GDP：国内生产总值（地域原则）
- GNP：国民生产总值（国民原则）

**通货膨胀与通货紧缩：**

| 类型 | 表现 | 对策 |
|------|------|------|
| 通胀 | 物价上涨、货币贬值 | 紧缩性政策 |
| 通缩 | 物价下降、货币升值 | 扩张性政策 |

**货币政策工具：**
- 存款准备金率
- 再贴现率
- 公开市场操作

### 四、历史人文

| 考点 | 内容 |
|------|------|
| 中国古代史 | 朝代更替、重大事件、制度变迁 |
| 中国近现代史 | 革命历程、重要人物 |
| 世界史 | 文艺复兴、工业革命、世界大战 |
| 文学艺术 | 名著、名画、名人 |

#### 高频考点

**朝代顺序：**
夏商周秦汉，魏晋南北朝，隋唐五代宋，元明清民国

**重要制度：**

|------|------|------|
| 分封制 | 西周 | 血缘宗法 |
| 郡县制 | 秦朝 | 中央集权 |
| 科举制 | 隋唐 | 选拔人才 |

**文学常识：**

| 作品 | 作者 | 朝代 |
|------|------|------|
| 《史记》 | 司马迁 | 西汉 |
| 《资治通鉴》 | 司马光 | 北宋 |
| 《红楼梦》 | 曹雪芹 | 清朝 |

### 五、地理常识

| 考点 | 内容 |
|------|------|
| 自然地理 | 地球运动、气候、地形 |
| 中国地理 | 行政区划、资源分布、名山大川 |
| 世界地理 | 大洲大洋、重要国家 |

#### 高频考点

**中国之最：**
- 最长河流：长江（6397km）
- 最大湖泊：青海湖（咸水湖）
- 最高山峰：珠穆朗玛峰

**气候类型：**

| 类型 | 特点 | 分布 |
|------|------|------|
| 热带季风 | 高温多雨 | 海南、台湾南部 |
| 亚热带季风 | 四季分明 | 长江中下游 |
| 温带季风 | 夏热冬冷 | 华北、东北 |

### 六、科技常识

| 考点 | 内容 |
|------|------|
| 基础科学 | 物理、化学、生物基础知识 |
| 前沿科技 | 人工智能、量子技术、航天科技 |
| 生活科技 | 日常科学现象解释 |

#### 高频考点

**航天成就：**
- 神舟系列：载人航天
- 嫦娥系列：探月工程
- 天问系列：火星探测
- 天宫空间站

**诺贝尔奖中国获得者：**
- 屠呦呦：生理学或医学奖（青蒿素）
- 莫言：文学奖

## 备考策略

### 1. 广泛涉猎

常识考查范围广，需要日常积累：
- 关注时事新闻
- 阅读科普文章
- 了解历史文化

### 2. 抓住重点

重点关注：
- 时政热点（近1-2年）
- 法律法规更新
- 重大科技成就

### 3. 排除法

遇到不确定的题目，运用排除法：
- 排除明显错误选项
- 结合常识判断

### 4. 合理取舍

常识题量大、分值低，建议：
- 快速作答，不纠结
- 会做的先做
- 不会的凭感觉选

::: tip 💡 备考建议
常识判断需要长期积累，建议每天花15-30分钟学习，重点关注时政热点和高频考点。
:::
