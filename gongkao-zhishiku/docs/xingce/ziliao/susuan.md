# 速算技巧

资料分析的关键在于"快"和"准"，掌握速算技巧是提高做题效率的核心。

## 核心速算方法

## 一、截位直除法

### 原理

将分子分母同时截取前几位有效数字，简化计算。

### 截位原则

| 选项差距 | 截位建议 |
|---------|---------|
| 选项首位不同 | 截2位 |
| 选项前两位不同 | 截3位 |
| 选项前三位相同 | 截4位或精算 |

### 示例

计算 $\frac{4856}{23147}$

**截位过程：**
$$\frac{4856}{23147} ≈ \frac{486}{2315} ≈ \frac{49}{232} ≈ 21\%$$

::: tip 💡 截位技巧
- 分子分母截位数量相同
- 四舍五入或同时放大/缩小
- 根据选项精度决定截位程度
:::

## 二、特殊分数速记

### 常用分数对照表

| 分数 | 百分比 | 分数 | 百分比 |
|------|--------|------|--------|
| 1/2 | 50% | 1/8 | 12.5% |
| 1/3 | 33.3% | 1/9 | 11.1% |
| 1/4 | 25% | 1/10 | 10% |
| 1/5 | 20% | 1/11 | 9.1% |
| 1/6 | 16.7% | 1/12 | 8.3% |
| 1/7 | 14.3% | 1/15 | 6.7% |

### 应用示例

计算 $\frac{2400}{7200}$

**速算：** $\frac{2400}{7200} = \frac{1}{3} = 33.3\%$

## 三、错位加减法

### 适用场景

乘法计算，将一个数拆分后分别计算再相加。

### 常用拆分

| 系数 | 拆分方式 |
|------|---------|
| ×1.1 | +10% |
| ×1.2 | +20% |
| ×0.9 | -10% |
| ×1.5 | +50% 或 ×3÷2 |
| ×1.25 | +25% 或 ×5÷4 |

### 示例

计算 $840 × 1.15$

**方法1：** $840 × 1.15 = 840 + 84 + 42 = 966$  
（+10%=84，+5%=42）

**方法2：** $840 × 1.15 ≈ 840 + 126 = 966$  
（+15% ≈ 840×0.15 = 126）

## 四、首数法

### 原理

快速计算结果的首位数字，用于排除选项。

### 适用场景

- 选项首位数字不同
- 需要快速排除干扰项

### 示例

$\frac{3847}{1926}$ 选项：A.1.5  B.2.0  C.2.5  D.3.0

**速判：** 3847 ≈ 2×1926，所以答案首位是2，选B。

## 五、尾数法

### 原理

通过计算结果的个位数字排除选项。

### 适用场景

- 加减乘法运算
- 选项尾数不同

### 示例

$247 + 358 + 469 =$ ？  
选项：A.1072  B.1074  C.1076  D.1078

**速判：** 7+8+9=24，尾数是4，选B。

## 六、比例法

### 原理

利用比例关系简化计算。

### 示例

已知A:B = 3:5，A+B = 160，求A。

**速算：** A = $\frac{3}{3+5}$ × 160 = $\frac{3}{8}$ × 160 = 60

## 七、增长率估算

### 增长率比较技巧

当比较多个增长率时：
- 增长量相近 → 基期小的增长率大
- 基期相近 → 增长量大的增长率大

### 增长率估算

$$r = \frac{增长量}{基期值} ≈ \frac{增长量}{现期值}（当r较小时）$$

## 综合应用

### 解题步骤

1. **看选项** - 确定精度要求
2. **选方法** - 根据计算类型选择速算方法
3. **快计算** - 大胆截位，快速得出结果
4. **验答案** - 必要时简单验证

### 实战示例

> 2024年某市进出口总额8456亿元，同比增长12.3%。求2023年进出口总额。

**分析：** 求基期值，公式为 $\frac{现期值}{1+r}$

**速算：**
$$\frac{8456}{1.123} ≈ \frac{8456}{1.12} ≈ \frac{846}{112} ≈ 7.5 \times 1000 = 7500亿元$$

## 速算口诀

::: tip 💡 速算记忆口诀
- 截位直除选项定，两位三位看精度
- 特殊分数要牢记，遇到直接写答案
- 错位加减算乘法，拆成整数更简单
- 首数尾数排选项，一眼看出正确项
:::
