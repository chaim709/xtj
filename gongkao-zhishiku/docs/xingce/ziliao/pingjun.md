# 平均数计算

平均数是资料分析的重要考点，涉及现期平均数、基期平均数和平均数的增长率。

## 基本概念

### 什么是平均数？

**平均数** = 总量 / 个数

::: info 📖 常见类型
- **人均**：总量 / 人口数
- **均价**：总金额 / 数量
- **平均增速**：与复利类似的增长概念
:::

## 核心公式

### 1. 现期平均数

$$平均数 = \frac{总量}{个数}$$

### 2. 基期平均数

$$基期平均数 = \frac{总量/(1+a)}{个数/(1+b)} = \frac{总量}{个数} \times \frac{1+b}{1+a}$$

其中 a 为总量增长率，b 为个数增长率

### 3. 平均数增长率

$$r_{平均} = \frac{a-b}{1+b}$$

其中 a 为总量增长率，b 为个数增长率

::: tip 💡 记忆技巧
平均数增长率公式与比重变化量公式类似：
- 分子：总量增长率 - 个数增长率
- 分母：1 + 个数增长率
:::

### 4. 平均数变化判断

- 总量增长率 > 个数增长率 → 平均数上升
- 总量增长率 < 个数增长率 → 平均数下降
- 总量增长率 = 个数增长率 → 平均数不变

## 典型例题

### 例1：现期平均数

> 2024年某公司销售额为3600万元，销售商品12万件，求平均每件商品售价。

**解答：**
$$平均价格 = \frac{3600}{12} = 300元$$

### 例2：基期平均数

> 2024年某市人均GDP为8万元，GDP同比增长10%，人口同比增长2%。求2023年人均GDP。

**解答：**
$$基期人均GDP = 8 \times \frac{1+2\%}{1+10\%} = 8 \times \frac{1.02}{1.10} ≈ 7.42万元$$

### 例3：平均数增长率

> 2024年某工厂产值同比增长15%，职工人数同比增长5%。求人均产值的增长率。

**解答：**
$$r_{人均} = \frac{15\% - 5\%}{1 + 5\%} = \frac{10\%}{1.05} ≈ 9.5\%$$

### 例4：平均数变化判断

> 2024年进口总额同比增长8%，进口商品数量同比增长12%。问进口商品的均价与去年相比如何变化？

**解答：**
总额增长率（8%）< 数量增长率（12%），所以均价**下降**。

## 速算技巧

### 1. 估算法

平均数增长率 ≈ 总量增长率 - 个数增长率（当个数增长率较小时）

### 2. 变化判断优先

很多题目只需要判断升降，不需要精确计算。

### 3. 与比重类比

平均数和比重的公式结构相似，可以类比记忆。

## 特殊情况

### 年均增长率

如果n年内从A增长到B，年均增长率为：

$$r = \sqrt[n]{\frac{B}{A}} - 1$$

::: warning ⚠️ 注意
年均增长率计算涉及开方，考试中通常给出近似值或可以用选项验证。
:::

## 易错点总结

::: warning ⚠️ 常见错误
1. **混淆分子分母**
   - 平均数 = 总量/个数（不是个数/总量）

2. **增长率使用错误**
   - 基期平均数：乘以 (1+b)/(1+a)
   - 注意 a 是总量增长率，b 是个数增长率

3. **平均数增长率 vs 平均增长率**
   - 平均数增长率：使用公式 (a-b)/(1+b)
   - 平均增长率：多期增长率的平均
:::
