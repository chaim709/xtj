# 比重计算

比重是资料分析中的核心考点，几乎每套试卷必考2-3道。

## 基本概念

### 什么是比重？

**比重** = 部分 / 整体 × 100%

::: info 📖 术语说明
- **部分**：某一类别或某一时期的数值
- **整体**：总量或总体数值
- **比重变化**：两个时期比重的差值（单位：百分点）
:::

## 核心公式

### 1. 现期比重

$$比重 = \frac{部分值}{整体值} \times 100\%$$

### 2. 基期比重

$$基期比重 = \frac{部分值/(1+a)}{整体值/(1+b)} = \frac{部分值}{整体值} \times \frac{1+b}{1+a}$$

其中 a 为部分增长率，b 为整体增长率

### 3. 比重变化（升降判断）

::: tip 💡 核心规律
- 部分增长率 > 整体增长率 → 比重上升
- 部分增长率 < 整体增长率 → 比重下降
- 部分增长率 = 整体增长率 → 比重不变
:::

### 4. 比重变化量

$$比重变化 = 现期比重 - 基期比重 = \frac{部分值}{整体值} \times \frac{a-b}{1+a}$$

## 典型例题

### 例1：现期比重计算

> 2024年某省农业产值2400亿元，全省GDP为12000亿元，求农业产值占GDP的比重。

**解答：**
$$比重 = \frac{2400}{12000} = 20\%$$

### 例2：基期比重计算

> 2024年某市工业产值8000亿元，同比增长10%；全市GDP为20000亿元，同比增长8%。求2023年工业产值占GDP的比重。

**解答：**
$$基期比重 = \frac{8000}{20000} \times \frac{1+8\%}{1+10\%} = 40\% \times \frac{1.08}{1.10} ≈ 39.3\%$$

### 例3：比重变化判断

> 2024年第三产业增加值同比增长12%，GDP同比增长8%。问第三产业占GDP的比重与去年相比是上升还是下降？

**解答：**
第三产业增长率（12%）> GDP增长率（8%），所以比重**上升**。

## 速算技巧

### 1. 直接观察法（判断升降）

只需比较部分和整体的增长率大小即可判断比重升降。

### 2. 估算法

比重变化量的绝对值一般不会太大（通常在5个百分点以内），可以快速估算。

### 3. 选项差距法

当选项差距较大时，可以粗略计算后排除。

## 易错点总结

::: warning ⚠️ 注意事项
1. **比重 vs 比值**
   - 比重：部分/整体，结果≤100%
   - 比值：A/B，可以大于1

2. **百分数 vs 百分点**
   - 比重本身用百分数表示（如30%）
   - 比重变化用百分点表示（如上升2个百分点）

3. **基期比重的计算**
   - 不要忘记乘以 (1+b)/(1+a)
:::
