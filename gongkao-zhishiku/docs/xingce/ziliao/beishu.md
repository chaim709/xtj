# 倍数计算

倍数是资料分析中的基础考点，计算相对简单，是拿分的关键题型。

## 基本概念

### 什么是倍数？

**倍数** = A / B

表示A是B的多少倍。

::: info 📖 相关概念
- **A是B的n倍**：A = n × B
- **A比B多n倍**：A = (1+n) × B
- **A是B的n%**：A = n% × B
:::

## 核心公式

### 1. 现期倍数

$$倍数 = \frac{A}{B}$$

### 2. 基期倍数

$$基期倍数 = \frac{A/(1+a)}{B/(1+b)} = \frac{A}{B} \times \frac{1+b}{1+a}$$

其中 a 为A的增长率，b 为B的增长率

### 3. 倍数与比值的关系

| 表述 | 含义 |
|------|------|
| A是B的2倍 | A/B = 2 |
| A比B多1倍 | A/B = 2 |
| A是B的200% | A/B = 2 |
| A比B多100% | A/B = 2 |

## 典型例题

### 例1：现期倍数

> 2024年甲公司产值800亿元，乙公司产值200亿元。甲公司产值是乙公司的多少倍？

**解答：**
$$倍数 = \frac{800}{200} = 4倍$$

### 例2：基期倍数

> 2024年A地区GDP为6000亿元，同比增长20%；B地区GDP为3000亿元，同比增长10%。2023年A地区GDP是B地区的多少倍？

**解答：**
$$基期倍数 = \frac{6000}{3000} \times \frac{1+10\%}{1+20\%} = 2 \times \frac{1.1}{1.2} ≈ 1.83倍$$

### 例3：多几倍问题

> 2024年甲公司利润是乙公司的3倍，问甲公司利润比乙公司多多少倍？

**解答：**
甲是乙的3倍，说明 甲 = 3×乙
甲比乙多 = 甲 - 乙 = 3乙 - 乙 = 2乙
所以甲比乙**多2倍**。

### 例4：百分比与倍数转换

> 甲是乙的150%，问甲是乙的多少倍？甲比乙多百分之多少？

**解答：**
- 甲是乙的 **1.5倍**（150% = 1.5）
- 甲比乙多 **50%**（150% - 100% = 50%）

## 速算技巧

### 1. 直接计算法

倍数计算通常比较简单，直接除法即可。

### 2. 截位法

分子分母同时截取前2-3位有效数字。

### 3. 选项代入验证

选项差距大时，估算后选最接近的。

## 易错点总结

::: warning ⚠️ 常见陷阱
1. **"是...倍" vs "多...倍"**
   - A是B的3倍：A = 3B
   - A比B多2倍：A = 3B（多2倍意味着A是B的3倍）

2. **基期倍数的计算**
   - 不要忘记乘以 (1+b)/(1+a)
   - 注意增长率对应的是哪个量

3. **倍数 vs 增长率**
   - 倍数：两个量的比值
   - 增长率：增长量与基期值的比值
:::

## 与其他概念的关系

| 概念 | 公式 | 特点 |
|------|------|------|
| 倍数 | A/B | 无单位 |
| 比重 | 部分/整体 | ≤100% |
| 增长率 | 增长量/基期值 | 可正可负 |
