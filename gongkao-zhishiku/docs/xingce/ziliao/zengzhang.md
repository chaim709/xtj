# 增长率计算

增长率是资料分析中最核心的考点之一，几乎每套试卷都会涉及。

## 基本概念

### 什么是增长率？

**增长率** = （现期值 - 基期值）/ 基期值 × 100%

简化公式：**增长率 = 增长量 / 基期值**

::: info 📖 术语说明
- **现期**：当前统计时期（题目描述的时间点）
- **基期**：参照对比的时期（通常是去年同期）
- **增长量**：现期值 - 基期值
:::

## 核心公式

### 1. 已知现期值和基期值，求增长率

$$r = \frac{现期值 - 基期值}{基期值} = \frac{现期值}{基期值} - 1$$

### 2. 已知现期值和增长率，求基期值

$$基期值 = \frac{现期值}{1 + r}$$

### 3. 已知基期值和增长率，求现期值

$$现期值 = 基期值 \times (1 + r)$$

### 4. 间隔增长率

如果两年的增长率分别为 $r_1$ 和 $r_2$，则两年间的综合增长率为：

$$r = r_1 + r_2 + r_1 \times r_2$$

::: tip 💡 速算技巧
当 $r_1$ 和 $r_2$ 都较小（< 10%）时，$r_1 \times r_2$ 可以忽略不计，直接用 $r ≈ r_1 + r_2$
:::

## 典型例题

### 例1：基础增长率计算

> 2024年某市GDP为8500亿元，2023年GDP为8000亿元，求2024年GDP增长率。

**解答：**
$$r = \frac{8500 - 8000}{8000} = \frac{500}{8000} = 6.25\%$$

### 例2：已知增长率求基期值

> 2024年某公司营收12亿元，同比增长20%，求2023年营收。

**解答：**
$$基期值 = \frac{12}{1 + 20\%} = \frac{12}{1.2} = 10亿元$$

### 例3：间隔增长率

> 2022年增长率为8%，2023年增长率为10%，求2021-2023年的总增长率。

**解答：**
$$r = 8\% + 10\% + 8\% \times 10\% = 18\% + 0.8\% = 18.8\%$$

## 速算技巧

### 截位直除法

当计算 $\frac{A}{B}$ 时，可以对分子分母同时截位：
- 保留2-3位有效数字
- 根据选项精度决定截位程度

**示例：** $\frac{4856}{23147}$ ≈ $\frac{486}{2315}$ ≈ $\frac{49}{232}$ ≈ 21%

### 特殊分数记忆

| 分数 | 百分比 | 分数 | 百分比 |
|------|--------|------|--------|
| 1/2 | 50% | 1/6 | 16.7% |
| 1/3 | 33.3% | 1/7 | 14.3% |
| 1/4 | 25% | 1/8 | 12.5% |
| 1/5 | 20% | 1/9 | 11.1% |

## 易错点总结

::: warning ⚠️ 注意区分
1. **"增长了"vs"增长到"**
   - 增长了10% → 现期 = 基期 × 1.1
   - 增长到110% → 现期 = 基期 × 1.1

2. **同比 vs 环比**
   - 同比：与去年同期相比
   - 环比：与上一个统计周期相比

3. **负增长**
   - 增长率为负数时，基期值 > 现期值
:::
