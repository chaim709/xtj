# 图形推理

图形推理考查空间想象和规律发现能力，需要建立图形敏感度。

## 题型分类

| 题型 | 占比 | 特点 |
|------|------|------|
| 规律推理 | 60% | 找平面图形规律 |
| 空间重构 | 30% | 六面体、展开图 |
| 分类分组 | 10% | 将图形分成两组 |

## 一、规律类题型

### 观察顺序

::: tip 💡 图形推理口诀
1. **元素组成相同** → 看位置（平移、旋转、翻转）
2. **元素组成相似** → 看样式（叠加、遍历）
3. **元素组成不同** → 看数量（点、线、面、角）
:::

### 1. 数量规律

#### 点的数量

| 类型 | 说明 |
|------|------|
| 交点 | 线与线的交叉点 |
| 切点 | 曲线相切的点 |
| 端点 | 线段的端点 |

#### 线的数量

| 类型 | 说明 |
|------|------|
| 直线数 | 直线段的数量 |
| 曲线数 | 曲线的数量 |
| 笔画数 | 一笔画完成的最少笔画 |

::: tip 💡 一笔画判断
- 奇点数 = 0 或 2 → 可以一笔画
- 奇点数 > 2 → 笔画数 = 奇点数/2
- **奇点**：从该点出发的线条数为奇数
:::

#### 面的数量

| 类型 | 说明 |
|------|------|
| 封闭区域 | 完全封闭的面 |
| 部分数 | 分割出的部分 |
| 特殊形状 | 三角形、正方形等 |

#### 角的数量

| 类型 | 说明 |
|------|------|
| 内角 | 封闭图形内部的角 |
| 锐角/直角/钝角 | 按角度分类 |

### 2. 位置规律

#### 平移

图形整体移动，方向和距离有规律。

#### 旋转

| 类型 | 特点 |
|------|------|
| 整体旋转 | 整个图形绕中心旋转 |
| 局部旋转 | 部分元素旋转 |

**常见旋转角度：** 45°、90°、180°

#### 翻转

| 类型 | 特点 |
|------|------|
| 左右翻转 | 沿垂直轴翻转 |
| 上下翻转 | 沿水平轴翻转 |

### 3. 样式规律

#### 叠加

| 类型 | 规则 |
|------|------|
| 直接叠加 | 所有元素保留 |
| 去同存异 | 相同部分消失 |
| 去异存同 | 只保留重叠部分 |
| 黑白叠加 | 黑+黑=白，黑+白=黑 |

#### 遍历

每个元素在每行/每列都出现一次。

### 4. 特殊规律

#### 对称性

| 类型 | 特点 |
|------|------|
| 轴对称 | 有对称轴 |
| 中心对称 | 绕中心点旋转180°重合 |

#### 封闭性与连通性

| 类型 | 特点 |
|------|------|
| 开放图形 | 有开口 |
| 封闭图形 | 无开口 |
| 连通图 | 所有部分相连 |

## 二、空间重构

### 1. 六面体折叠

**相对面判断方法：**

展开图中，相隔一个面的两个面是相对面。

**相邻面判断方法：**

找公共边，确定相邻关系。

### 2. 展开图还原

**解题步骤：**
1. 找特殊面（有文字、箭头等）
2. 确定相对面和相邻面
3. 根据选项逐一排除

### 3. 三视图

| 视图 | 方向 |
|------|------|
| 主视图 | 正面看 |
| 俯视图 | 从上往下看 |
| 左视图 | 从左边看 |

### 4. 截面图

想象一个平面切过立体图形，得到的截面形状。

## 三、分类分组

### 解题方法

找出6个图形可以分成的两组（每组3个），每组内部有共同特征。

### 常见分组依据

- 曲直性（直线/曲线）
- 对称性（对称/不对称）
- 封闭性（封闭/开放）
- 数量规律（相同数量分一组）

## 图形敏感度训练

### 看到以下特征时优先考虑：

| 图形特征 | 优先考虑 |
|---------|---------|
| 明显有角 | 角的数量 |
| 线条相交多 | 交点数量 |
| 多个独立部分 | 部分数、连通性 |
| 图形相似 | 旋转、翻转 |
| 黑白块 | 叠加规律 |
| 对称图形 | 对称性 |
| 田字格/九宫格 | 遍历、叠加 |

## 解题口诀

::: tip 💡 图形推理口诀
- 相同看位置，相似看样式
- 不同数个数，特殊看对称
- 六面体找相对，展开图看相邻
- 曲直分组是常见，封闭对称要敏感
:::
