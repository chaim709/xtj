# 🎨 判断推理

判断推理考查逻辑思维能力，是行测中技巧性最强的模块。

## 考情概述

| 考试类型 | 题量 | 分值 | 难度 |
|---------|------|------|------|
| 国考 | 40题 | 约28分 | ⭐⭐⭐⭐ |
| 省考 | 35-40题 | 约24分 | ⭐⭐⭐ |

## 四大题型

### 🖼️ 图形推理
- [图形推理技巧](./tuxing.md)
- 数量规律 - 点、线、面、角的数量
- 位置规律 - 平移、旋转、翻转
- 样式规律 - 叠加、遍历、对称
- 空间重构 - 六面体、展开图

### 📋 定义判断
- [定义判断技巧](./dingyi.md)
- 单定义判断 - 提取关键词匹配
- 多定义判断 - 对比区分定义

### 🔗 类比推理
- [类比推理技巧](./leibi.md)
- 语义关系 - 近义、反义、比喻
- 逻辑关系 - 全同、包含、交叉、并列
- 语法关系 - 主谓、动宾、偏正

### 🧠 逻辑判断
- [逻辑判断技巧](./luoji.md)
- 翻译推理 - 充分必要条件
- 组合排列 - 真假话、元素对应
- 加强削弱 - 论证分析
- 日常推理 - 归纳、演绎

## 核心技巧

::: tip 💡 图形推理口诀
- 看到对称想翻转
- 看到曲线数曲直
- 元素组成相似看位置
- 元素组成不同数个数
- 多图推理找叠加
:::

::: tip 💡 逻辑判断翻译公式
- 如果A那么B → A→B
- 只有A才B → B→A
- A除非B → 非B→A
- 否定一个推另一个，肯定一个推不出
:::

## 逻辑关系速查

| 原命题 | 逆否命题 | 关系 |
|--------|----------|------|
| A→B | 非B→非A | 等价 |
| A且B | 非A或非B | 矛盾 |
| A或B | 非A且非B | 矛盾 |
