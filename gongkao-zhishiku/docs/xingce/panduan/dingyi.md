# 定义判断

定义判断考查对概念的理解和应用能力，关键在于提取定义的核心要素。

## 题型特点

| 特征 | 说明 |
|------|------|
| 题干 | 给出一个概念的定义 |
| 问题 | 判断哪个选项符合/不符合定义 |
| 难度 | 相对简单，正确率较高 |

## 解题方法

### 核心思路

**提取关键信息 → 逐一对比选项 → 匹配or排除**

### 1. 提取关键信息

定义的关键信息通常包括：

| 要素 | 示例问题 |
|------|---------|
| 主体 | 谁？（个人、组织、政府等） |
| 客体 | 对谁？对什么？ |
| 原因/目的 | 为什么？为了什么？ |
| 条件 | 在什么情况下？ |
| 方式/手段 | 通过什么方式？ |
| 结果 | 产生什么结果？ |

### 2. 逐条对比

将选项与提取的关键信息逐一比对：
- 全部符合 → 正确选项
- 有一条不符合 → 排除

### 3. 排除法

当正面选择困难时，使用排除法：
- 明显不符合某一要素的选项先排除
- 最后对比剩余选项

## 例题解析

### 例1：单定义判断

> **功能固着**是指一个人看到某个物体的一种惯常用途后，就很难看出它的其他新用途。
>
> 下列属于功能固着的是：

**分析关键要素：**
- 主体：一个人
- 对象：某个物体
- 表现：习惯性认为只有一种用途，看不到新用途

**选项判断：**
- A. 用砖头垫桌腿 ❌（发现了新用途，不是固着）
- B. 认为眼镜只能用来看东西 ✅（符合定义）
- C. 用报纸包东西 ❌（发现了新用途）
- D. 学习新技能 ❌（与物体用途无关）

### 例2：多定义判断

> **显性知识**是指可以用文字、图表等方式表达的知识。
> **隐性知识**是指不易用语言表达、依赖个人经验的知识。
>
> 下列属于隐性知识的是：

**分析两个定义的区别：**
- 显性知识：可以表达、可以传授
- 隐性知识：难以表达、依赖经验

**答案：** 选择"骑自行车的技巧"（需要亲身体验，难以用语言完全表达）

## 常见陷阱

### 1. 偷换主体

选项的主体与定义不一致。

### 2. 偷换目的

选项的目的与定义要求不符。

### 3. 扩大/缩小范围

选项的范围超出或小于定义的限定。

### 4. 无中生有

选项包含定义中没有的内容。

### 5. 必要条件遗漏

选项缺少定义中的必要条件。

## 解题技巧

### 1. 先读问题

确定是选"符合"还是"不符合"定义的选项。

### 2. 划关键词

在定义中划出核心要素，便于对比。

### 3. 一一对应

选项的每个要素都要与定义对应。

### 4. 合理推断

有些概念需要结合常识理解。

### 5. 同类排除

如果两个选项意思相近，通常都不是答案。

## 高频定义类型

| 类型 | 示例 |
|------|------|
| 心理学概念 | 晕轮效应、刻板印象、从众心理 |
| 经济学概念 | 沉没成本、机会成本、边际效应 |
| 法律概念 | 正当防卫、紧急避险、不当得利 |
| 社会学概念 | 社会化、角色冲突、文化认同 |
| 管理学概念 | 激励机制、组织结构、决策模型 |

## 解题口诀

::: tip 💡 定义判断口诀
- 先看问题问什么
- 提取定义关键词
- 主体客体和条件
- 目的方式加结果
- 选项逐一来比对
- 一条不符就排除
:::
