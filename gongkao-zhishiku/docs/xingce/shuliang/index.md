## 经典公式

### 行程问题
- 路程 = 速度 × 时间
- 相遇：S = (V₁ + V₂) × t
- 追及：S = (V₁ - V₂) × t

### 工程问题
- 工作量 = 效率 × 时间
- 设总工作量为最小公倍数

### 利润问题
- 利润 = 售价 - 成本
- 利润率 = 利润 / 成本 × 100%

## 备考建议

::: warning ⚠️ 数量关系备考策略
1. **优先掌握高频题型** - 工程、行程、利润、排列组合

3. **刷题积累题感** - 熟悉常见题型的解题套路
4. **控制时间** - 每题不超过2分钟
:::
