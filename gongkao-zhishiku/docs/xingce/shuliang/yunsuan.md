## 一、工程问题

### 核心公式

$$工作量 = 效率 \times 时间$$

### 赋值法

**设工作总量为时间的最小公倍数**

**例题：** 甲单独完成需要10天，乙单独完成需要15天，两人合作需要多少天？

**解答：**
- 设总工作量 = LCM(10,15) = 30
- 甲效率 = 30/10 = 3
- 乙效率 = 30/15 = 2
- 合作效率 = 3+2 = 5
- 合作时间 = 30/5 = **6天**

### 常见变形

| 类型 | 特点 |
|------|------|
| 交替合作 | 按周期计算 |
| 水池问题 | 有进有出 |
| 轮流工作 | 注意衔接 |

## 二、行程问题

### 基本公式

$$路程 = 速度 \times 时间$$

### 常见模型

#### 1. 相遇问题

$$S_{相遇} = (V_1 + V_2) \times t$$

#### 2. 追及问题

$$S_{追及} = (V_1 - V_2) \times t$$

#### 3. 流水行船

- 顺流速度 = 船速 + 水速
- 逆流速度 = 船速 - 水速

#### 4. 环形相遇

- 同向追及一圈：$t = \frac{周长}{V_1 - V_2}$
- 相向相遇：$t = \frac{周长}{V_1 + V_2}$

**例题：** 甲乙两地相距300km，A车从甲出发速度60km/h，B车从乙出发速度40km/h，相向而行，几小时相遇？

**解答：**
$$t = \frac{300}{60+40} = 3小时$$

## 三、经济利润问题

### 核心公式

$$利润 = 售价 - 成本$$
$$利润率 = \frac{利润}{成本} \times 100\%$$
$$售价 = 成本 \times (1 + 利润率)$$

### 特值法

设成本为100，方便计算。

**例题：** 某商品进价100元，标价150元，打8折出售，求利润率。

**解答：**
- 成本 = 100元
- 售价 = 150 × 0.8 = 120元
- 利润 = 120 - 100 = 20元
- 利润率 = 20/100 = **20%**

### 常见变形

| 类型 | 关键点 |
|------|--------|
| 打折销售 | 注意标价和售价的区别 |
| 分段定价 | 分类讨论 |
| 成本变化 | 明确哪个量变化 |

## 四、排列组合

### 基本概念

- **排列**：有顺序，用 $A_n^m = \frac{n!}{(n-m)!}$
- **组合**：无顺序，用 $C_n^m = \frac{n!}{m!(n-m)!}$

### 常用公式

$$A_n^m = n \times (n-1) \times ... \times (n-m+1)$$
$$C_n^m = \frac{A_n^m}{m!}$$

### 解题原则

::: tip 💡 核心原则
- **分类用加法**：完成一件事有多种方法，各方法相加
- **分步用乘法**：完成一件事分多步，各步相乘
:::

## 五、概率问题

### 基本公式

$$P = \frac{满足条件的情况数}{所有可能的情况数}$$

### 常见类型

#### 1. 古典概型

$$P(A) = \frac{n(A)}{n(S)}$$

#### 2. 条件概率

$$P(A|B) = \frac{P(AB)}{P(B)}$$

#### 3. 独立事件

$$P(AB) = P(A) \times P(B)$$

## 六、几何问题

### 常用公式

#### 平面图形

| 正方形 | 4a | a² |

| 长方体 | 2(ab+bc+ac) | abc |

| 圆锥 | πr²+πrl | ⅓πr²h |
| 球 | 4πr² | ⅔πr³ |

## 七、容斥问题

### 两集合

$$|A∪B| = |A| + |B| - |A∩B|$$

### 三集合

$$|A∪B∪C| = |A| + |B| + |C| - |A∩B| - |B∩C| - |A∩C| + |A∩B∩C|$$
