# 逻辑填空

逻辑填空考查词语辨析和语境分析能力，需要大量积累和技巧结合。

## 题型概述

| 类型 | 占比 | 特点 |
|------|------|------|
| 实词填空 | 40% | 近义词辨析 |
| 成语填空 | 35% | 成语积累+语境 |
| 混合填空 | 25% | 实词+成语组合 |

## 一、语境分析法

### 1. 解释关系

**特征：** 空缺处与其他内容互相解释

**标志词：**
- 即、就是、可以说、也就是说
- 冒号、破折号
- 无标志（前后句意思相近）

::: tip 💡 解题方法
找到解释说明的内容，选择意思相近的词语。
:::

### 2. 反对关系

**特征：** 空缺处与其他内容形成对比或转折

**标志词：**
- 转折：但是、然而、却、不过
- 对比：不是...而是、与其...不如
- 否定：并非、不是

::: tip 💡 解题方法
找到对立的内容，选择意思相反或相对的词语。
:::

### 3. 递进关系

**特征：** 语义程度逐层加深

**标志词：**
- 更、甚至、乃至、何况
- 不仅...而且、不但...还

::: tip 💡 解题方法
递进后语义更重，根据位置判断程度轻重。
:::

### 4. 并列关系

**特征：** 前后语义相近或相关

**标志词：**
- 和、与、同时、既...也
- 逗号、顿号分隔的并列成分

## 二、词语辨析法

### 1. 语义侧重

| 词语 | 侧重点 |
|------|-------|
| 推脱 | 推卸责任 |
| 推托 | 借故拒绝 |
| 推诿 | 把责任推给别人 |

| 词语 | 侧重点 |
|------|-------|
| 体味 | 品味、体会（抽象） |
| 体验 | 亲身经历 |
| 体察 | 观察了解 |

### 2. 搭配对象

| 词语 | 常见搭配 |
|------|---------|
| 发挥 | 作用、优势、特长 |
| 发扬 | 传统、精神、作风 |
| 发展 | 经济、事业、关系 |

| 词语 | 常见搭配 |
|------|---------|
| 坚定 | 信念、立场、态度 |
| 坚决 | 反对、执行、贯彻 |
| 坚持 | 原则、方向、锻炼 |

### 3. 感情色彩

| 类型 | 示例 |
|------|------|
| 褒义 | 成果、硕果、结晶 |
| 贬义 | 后果、恶果 |
| 中性 | 结果、效果 |

### 4. 程度轻重

| 程度 | 词语示例 |
|------|---------|
| 轻 | 批评、指责 |
| 重 | 谴责、声讨 |

## 三、高频成语辨析

### 易混成语（正确用法）

| 成语 | 含义 | 易错点 |
|------|------|-------|
| 望其项背 | 能够赶上 | 常误用为"赶不上" |
| 首当其冲 | 首先受到攻击 | 常误用为"首先、领先" |
| 差强人意 | 大体令人满意 | 常误用为"不满意" |
| 不以为然 | 不认为是对的 | 常误用为"不在意" |
| 不以为意 | 不放在心上 | 与上词混淆 |
| 耳濡目染 | 经常接触而受影响 | 强调无意识 |
| 潜移默化 | 不知不觉中受影响 | 与上词类似 |
| 一蹴而就 | 一下子成功 | 多用于否定句 |
| 一挥而就 | 一下子完成（写作等） | 与上词混淆 |
| 不可理喻 | 不能用道理使明白 | 常误用为"不可理解" |

### 近义成语辨析

| 成语A | 成语B | 区别 |
|-------|-------|------|
| 侃侃而谈 | 夸夸其谈 | 前者褒义，后者贬义 |
| 处心积虑 | 殚精竭虑 | 前者贬义，后者褒义 |
| 无所不至 | 无微不至 | 前者贬义，后者褒义 |
| 耸人听闻 | 骇人听闻 | 前者编造，后者真实 |
| 不胫而走 | 不翼而飞 | 前者指消息，后者指东西丢失 |

## 四、解题步骤

```
1. 通读文段，理解大意
       ↓
2. 分析空缺处语境关系
       ↓
3. 词语辨析（语义、搭配、色彩）
       ↓
4. 代入验证，确定答案
```

## 解题口诀

::: tip 💡 逻辑填空口诀
- 语境分析是根本
- 解释反对和递进
- 词语辨析看三点
- 语义搭配加色彩
- 成语积累不能少
- 代入检验语感好
:::
