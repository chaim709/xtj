# 语句表达

语句表达考查语言组织和表达能力，主要包括语句排序和语句填空两种题型。

## 一、语句排序

### 题型特征

给出若干打乱顺序的句子，要求排列成语意连贯的语段。

### 解题方法

#### 1. 确定首句

**不能做首句的情况：**

| 类型 | 示例 |
|------|------|
| 代词开头 | 这、那、此、其、他 |
| 关联词后半部分 | 但是、所以、然而、因此 |
| 总结性词语 | 总之、综上所述、由此可见 |
| 指示不明的内容 | 需要前文铺垫的句子 |

**可以做首句的情况：**

| 类型 | 示例 |
|------|------|
| 背景铺垫 | 随着...的发展 |
| 引入话题 | 时间、地点、人物交代 |
| 定义说明 | XXX是指... |
| 提出问题 | 问句引入 |

#### 2. 找关联词配对

| 前半部分 | 后半部分 |
|---------|---------|
| 虽然、尽管 | 但是、却、然而 |
| 因为、由于 | 所以、因此 |
| 不仅、不但 | 而且、还、更 |
| 如果、假如 | 那么、就 |

#### 3. 找指代关系

代词（这、那、此、其）必须有所指代，指代内容应在前文出现。

#### 4. 找逻辑关系

| 关系类型 | 表现 |
|---------|------|
| 时间顺序 | 先...后...、首先...然后... |
| 空间顺序 | 从...到...、由近及远 |
| 逻辑顺序 | 因果、总分、递进 |
| 主次顺序 | 主要...次要... |

#### 5. 验证法

将答案按顺序读一遍，检查是否通顺连贯。

### 例题解析

> 将以下句子排列成语意连贯的语段：
> ①但是，书法艺术的发展绝不是孤立的
> ②它与社会环境、文化思潮有密切关系
> ③书法是中华民族的艺术瑰宝
> ④因此，研究书法史必须结合文化史来进行
> ⑤每个时代的书法风格都带有时代的印记

**解析：**
- ①以"但是"开头，不能做首句，排除①开头选项
- ③引入话题"书法"，适合做首句
- ①②⑤都在论述书法与其他因素的关系，相互关联
- ④以"因此"总结，应该放最后

**答案：** ③①②⑤④

## 二、语句填空

### 题型特征

在语段中留出空白，要求填入恰当的句子。

### 填空位置分析

| 位置 | 作用 |
|------|------|
| 段首 | 引出话题、总领全文 |
| 段中 | 承上启下、过渡衔接 |
| 段尾 | 总结全文、得出结论 |

### 解题方法

#### 1. 话题一致

填入句子的主题必须与上下文一致。

#### 2. 结构对应

注意与前后句子的结构形式相呼应。

#### 3. 逻辑衔接

填入句子要与上下文形成合理的逻辑关系。

#### 4. 感情色彩

保持与语段整体感情基调一致。

### 例题解析

> 心理学研究表明，人们往往高估自己的能力，同时低估任务的难度。___________。因此，在制定计划时，应该预留充足的时间余量。

**选项：**
A. 这被称为"规划谬误"
B. 这说明人们缺乏自知之明
C. 因此人们常常无法按时完成任务
D. 这种现象在年轻人中更为普遍

**解析：**
- 空格位于段中，起承上启下作用
- 上文说明问题现象，下文说"因此"给出对策
- C选项说明这种现象导致的后果，能够衔接下文的对策
- A只是下定义，B和D都偏离主线

**答案：** C

## 三、下文推断

### 题型特征

- 作者接下来最可能讲述的是...
- 这段文字之后最可能论述的是...

### 解题方法

#### 1. 看尾句

重点关注最后一句话的内容，下文应该承接尾句展开。

#### 2. 排除已述内容

前文已经详细论述的内容，后文不会重复。

#### 3. 保持话题一致

下文的主题应该与尾句话题一致。

### 例题解析

> 近年来，人工智能技术发展迅速，在医疗、交通、教育等领域得到广泛应用。然而，人工智能在发展中也面临着许多挑战，如数据安全、算法偏见等问题。专家指出，解决这些问题需要政府、企业和社会共同努力。

**问：作者接下来最可能论述的是？**

A. 人工智能的发展历史
B. 政府、企业、社会各方应采取的具体措施
C. 人工智能在各领域的应用案例
D. 人工智能带来的社会问题

**解析：**
- 尾句提到"需要政府、企业和社会共同努力"
- 下文应该承接这个话题，论述具体如何努力
- A、C是已述或无关内容，D与尾句方向相反

**答案：** B

## 解题口诀

::: tip 💡 语句表达口诀
- 排序先看首尾句
- 代词指代要明确
- 关联词语要配对
- 填空话题要一致
- 下文推断看尾句
- 验证通读保通顺
:::
