# 📖 言语理解与表达

言语理解与表达是行测中题量最大的模块，主要考查对语言文字的理解和运用能力。

## 考情概述

| 考试类型 | 题量 | 分值 | 难度 |
|---------|------|------|------|
| 国考 | 40题 | 约32分 | ⭐⭐⭐⭐ |
| 省考 | 30-40题 | 约24分 | ⭐⭐⭐ |

## 三大题型

### 📄 片段阅读
- [片段阅读技巧](./pianduan.md)
- 主旨概括题 - 找准文段核心观点
- 意图判断题 - 理解作者言外之意
- 细节理解题 - 精准定位信息对应
- 标题添加题 - 概括+吸引原则

### ✏️ 逻辑填空
- [逻辑填空技巧](./tiankong.md)
- 实词辨析 - 语义+搭配+语境
- 成语填空 - 积累+语境分析
- 混合填空 - 综合运用技巧

### 💬 语句表达
- [语句表达技巧](./yuju.md)
- 语句排序 - 找首尾+看关联
- 语句填空 - 承上启下+话题一致
- 下文推断 - 尾句分析

## 高频考点

::: tip 💡 片段阅读核心技巧
1. **关联词定位法** - 转折、递进、因果后是重点
2. **行文脉络分析** - 总分、分总、总分总结构
3. **主体排除法** - 选项主体与文段不符直接排除
:::

::: tip 💡 逻辑填空核心技巧
1. **语境分析** - 解释关系、反对关系、递进关系
2. **词语辨析** - 语义侧重、搭配对象、感情色彩
3. **验证代入** - 最后将答案代入检验语感
:::

## 易错词语积累

| 词语A | 词语B | 区别 |
|-------|-------|------|
| 侧目 | 刮目 | 侧目=害怕；刮目=另眼相看 |
| 望其项背 | 望尘莫及 | 望其项背=能赶上；望尘莫及=赶不上 |
| 首当其冲 | 一马当先 | 首当其冲=首先受害；一马当先=领先 |
| 不以为然 | 不以为意 | 不以为然=不认同；不以为意=不在意 |
