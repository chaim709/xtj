# 公文写作

公文写作考查应用文的撰写能力，需要掌握不同文种的格式和内容要求。

## 题型概述

| 特征 | 说明 |
|------|------|
| 分值 | 20-25分 |
| 字数 | 400-600字 |
| 难度 | ⭐⭐⭐ |
| 核心 | 格式正确 + 内容全面 |

## 常见文种

| 类别 | 文种 | 特点 |
|------|------|------|
| 宣传类 | 倡议书、宣传稿、公开信 | 号召性强 |
| 建议类 | 建议书、意见、提案 | 针对问题提对策 |
| 汇报类 | 调研报告、工作汇报、情况报告 | 客观全面 |
| 方案类 | 工作方案、活动策划、实施方案 | 具体可行 |
| 发言类 | 讲话稿、发言稿、致辞 | 场合感强 |
| 评论类 | 短评、编者按 | 观点鲜明 |

## 公文格式

### 标准格式

```
                    标题（居中）
                    
主送单位：（或称呼）

    正文（首行缩进2字符）
    
    第一部分...
    
    第二部分...
    
    第三部分...
    
    结尾...
                    
                              发文单位（右对齐）
                              日期（右对齐）
```

### 格式要素

| 要素 | 要求 | 是否必须 |
|------|------|---------|
| 标题 | 居中，概括主题 | ✅ |
| 称呼 | 顶格，加冒号 | 视文种而定 |
| 正文 | 首行缩进 | ✅ |
| 落款 | 单位+日期，右对齐 | 视文种而定 |

## 各文种写作要点

### 一、倡议书

**格式：** 标题 + 称呼 + 正文 + 落款

**结构：**
```
标题：关于XXX的倡议书

[称呼]：

    [背景说明]
    
    [倡议内容]
    一、...
    二、...
    三、...
    
    [号召语]
    
                              倡议单位
                              X年X月X日
```

**语言特点：** 有感染力、号召力

### 二、建议书

**格式：** 标题 + 称呼 + 正文 + 落款

**结构：**
```
标题：关于XXX的建议

[称呼]：

    [问题说明]
    
    [具体建议]
    一、...
    二、...
    三、...
    
    [期望表态]
    
                              建议单位
                              X年X月X日
```

**语言特点：** 条理清晰、措施具体

### 三、调研报告

**格式：** 标题 + 正文（无称呼、无落款）

**结构：**
```
标题：关于XXX的调研报告

一、调研背景/目的

二、调研方法/过程

三、调研发现/问题
    （一）...
    （二）...
    
四、意见建议/对策
    （一）...
    （二）...
```

**语言特点：** 客观、全面、有数据支撑

### 四、讲话稿

**格式：** 标题 + 称呼 + 正文（无落款）

**结构：**
```
标题：在XXX会议上的讲话

[称呼]：

    [开场语]
    
    [主体内容]
    一、...
    二、...
    三、...
    
    [结束语]
    
    谢谢大家！
```

**语言特点：** 口语化、有感染力

### 五、短评

**格式：** 标题 + 正文（无称呼、无落款）

**结构：**
```
标题：XXX（评论性标题）

    [引出话题]
    
    [展开评论]
    
    [总结升华]
```

**语言特点：** 观点鲜明、论证有力

## 写作技巧

### 1. 明确目的

| 文种 | 核心目的 |
|------|---------|
| 倡议书 | 号召行动 |
| 建议书 | 解决问题 |
| 调研报告 | 反映情况 |
| 讲话稿 | 动员部署 |

### 2. 身份代入

考虑自己的身份和受众：
- 政府官员 → 群众
- 单位负责人 → 员工
- 社区工作者 → 居民

### 3. 格式规范

- 根据文种确定是否需要称呼、落款
- 标题要准确反映内容
- 正文结构要清晰

### 4. 语言得体

| 场合 | 语言风格 |
|------|---------|
| 正式公文 | 严谨、规范 |
| 宣传类 | 生动、有感染力 |
| 汇报类 | 客观、准确 |
| 发言类 | 口语化、亲切 |

## 常见错误

::: warning ⚠️ 避免以下问题
1. **格式错误**：缺少必要格式要素
2. **身份错位**：语气与身份不符
3. **内容空洞**：缺乏材料支撑
4. **结构混乱**：缺乏逻辑层次
5. **语言不当**：过于口语化或过于书面化
:::

## 配套练习

建议结合历年真题进行练习，注意：
- 不同文种的格式差异
- 结合题目要求确定内容重点
- 语言风格要与文种匹配
