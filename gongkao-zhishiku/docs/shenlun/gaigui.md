# 概括归纳

概括归纳是申论的基础题型，考查从材料中提炼核心信息的能力。

## 题型概述

| 特征 | 说明 |
|------|------|
| 分值 | 10-15分 |
| 字数 | 150-300字 |
| 难度 | ⭐⭐ |
| 核心 | 材料为王，忠于原文 |

## 常见问法

| 问法 | 归纳对象 |
|------|---------|
| 概括/归纳主要问题 | 问题 |
| 概括主要原因 | 原因 |
| 概括主要做法/经验 | 做法/对策 |
| 概括主要影响/作用 | 影响 |
| 概括主要特点/特征 | 特点 |
| 概括主要变化 | 变化 |

## 解题步骤

```
1. 审题 - 明确归纳对象（问题/原因/做法/影响）
       ↓
2. 阅读材料 - 勾画相关内容
       ↓
3. 提炼要点 - 去除修饰，保留核心
       ↓
4. 分类整合 - 合并同类项
       ↓
5. 规范作答 - 条理清晰，标序号
```

## 答题结构

### 基本格式

```
总括句：（可选，高度概括所有要点）

一、XXX。具体表现为...
二、XXX。具体表现为...
三、XXX。具体表现为...
```

### 示例

**题目：** 请根据给定材料，概括当前农村教育存在的主要问题。（15分，不超过200字）

**答案：**

```
当前农村教育主要存在以下问题：

一、师资力量薄弱。教师数量不足，年龄老化，学历偏低，缺乏专业培训。
二、教学条件落后。校舍陈旧，教学设备短缺，信息化水平低。
三、学生流失严重。辍学率较高，留守儿童问题突出，学习动力不足。
四、教育投入不足。经费紧张，基础设施建设滞后。
```

## 核心技巧

### 1. 材料为王

::: tip 💡 核心原则
- 答案全部来自材料
- 不需要自己发挥
- 原文表述优先
:::

### 2. 关注标志词

| 类型 | 标志词 |
|------|--------|
| 问题 | 不足、缺乏、困难、矛盾、但是、然而 |
| 原因 | 由于、因为、根源在于、导致 |
| 做法 | 采取、实施、通过、建立、完善 |
| 影响 | 使得、促进、有利于、导致 |

### 3. 提炼方法

| 方法 | 说明 |
|------|------|
| 直接摘抄 | 材料中有现成表述 |
| 同义替换 | 将口语化表述转为书面语 |
| 综合概括 | 将多处信息整合为一点 |

### 4. 分类整合

- 将相似要点合并
- 按照主体、环节等分类
- 使用"一、二、三"分点

## 常见错误

::: warning ⚠️ 避免以下问题
1. **脱离材料**：自己发挥，没有材料依据
2. **要点遗漏**：没有全面覆盖材料内容
3. **重复累赘**：同一要点反复出现
4. **层次混乱**：缺少分类整合
5. **语言啰嗦**：没有精简提炼
6. **格式不规范**：没有分点标序号
:::

## 不同类型要点提炼

### 归纳问题

**材料原文：** 
> 调查发现，很多社区的垃圾分类设施不完善，居民不知道如何分类，分类后的垃圾又被混合收运。

**提炼要点：**
- 分类设施不完善
- 居民缺乏分类知识
- 收运环节混合处理

### 归纳原因

**材料原文：**
> 由于监管不到位，部分商家为追求利润，使用劣质原材料。

**提炼要点：**
- 监管不到位
- 商家逐利心理

### 归纳做法

**材料原文：**
> 该市建立了完善的网格化管理体系，每个网格配备专职网格员，实现了问题的及时发现和处理。

**提炼要点：**
- 建立网格化管理体系
- 配备专职网格员
- 实现问题及时发现处理

## 字数控制技巧

| 情况 | 处理方法 |
|------|---------|
| 字数不够 | 适当展开具体内容 |
| 字数超出 | 精简语句，删除修饰语 |
| 刚好合适 | 检查是否遗漏要点 |
