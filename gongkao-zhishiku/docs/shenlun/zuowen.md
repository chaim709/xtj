# 大作文

大作文是申论的压轴题，考查综合分析和文字表达能力，分值占比最大。

## 题型概述

| 特征 | 说明 |
|------|------|
| 分值 | 35-40分 |
| 字数 | 1000-1200字 |
| 难度 | ⭐⭐⭐⭐⭐ |
| 核心 | 立意准确 + 论证充分 |

## 常见命题形式

| 形式 | 示例 |
|------|------|
| 话题作文 | 请围绕"创新驱动发展"写一篇文章 |
| 标题作文 | 以"让城市更有温度"为题 |
| 引言作文 | 结合材料中某句话，写一篇文章 |

## 五段三分式结构

```
第一段：开头（引出话题 + 提出论点）
         100-150字
              ↓
第二段：分论点一 + 论证
         200-250字
              ↓
第三段：分论点二 + 论证
         200-250字
              ↓
第四段：分论点三 + 论证
         200-250字
              ↓
第五段：结尾（总结升华）
         100-150字
```

## 一、立意

### 立意要求

| 要求 | 说明 |
|------|------|
| 准确 | 紧扣材料主题，不跑题 |
| 明确 | 观点清晰，态度鲜明 |
| 深刻 | 有一定思想深度 |

### 立意方法

1. **抓材料核心观点**
   - 领导讲话
   - 专家观点
   - 政策文件

2. **分析材料逻辑**
   - 问题 → 原因 → 对策
   - 现象 → 本质 → 意义

3. **结合时代背景**
   - 国家战略
   - 社会热点

### 分论点设定

**常见模式：**

| 模式 | 示例 |
|------|------|
| 是什么+为什么+怎么办 | 内涵-意义-路径 |
| 并列式 | 三个方面/三个层面 |
| 递进式 | 个人-社会-国家 |
| 对比式 | 正面-反面 |

**示例：**

主题：创新驱动发展

```
分论点一：创新是引领发展的第一动力（意义）
分论点二：创新需要营造良好的制度环境（条件）
分论点三：创新要坚持以人民为中心（方向）
```

## 二、开头

### 开头模式

#### 1. 引言式

引用名言、政策、材料原文开头。

```
"科技是第一生产力，创新是引领发展的第一动力。"在新发展阶段，创新已成为推动高质量发展的核心引擎。然而，当前我国创新能力仍存在短板，关键核心技术受制于人的局面尚未根本改变。因此，必须把创新摆在国家发展全局的核心位置，加快建设创新型国家。
```

#### 2. 背景式

从时代背景、社会现象入手。

```
当今世界正经历百年未有之大变局，新一轮科技革命和产业变革蓬勃发展。与此同时，国际竞争日趋激烈，科技创新成为国际战略博弈的主战场。面对新形势新挑战，我们必须坚持创新驱动发展战略，把科技自立自强作为国家发展的战略支撑。
```

#### 3. 案例式

从具体案例引入。

```
从"嫦娥"探月到"天问"探火，从"北斗"组网到"奋斗者"号深潜万米海底，一项项重大科技成果令世界瞩目。这些成就的取得，正是我国坚持创新驱动发展战略的生动体现。创新，已成为推动中国高质量发展的强大引擎。
```

### 开头要点

::: tip 💡 开头三要素
1. **引出话题**：点明文章主题
2. **提出问题**：说明现状或问题
3. **亮明观点**：明确中心论点
:::

## 三、论证

### 论证结构

```
分论点（中心句）
       ↓
道理论证（为什么）
       ↓
事例论证（举例说明）
       ↓
回扣分论点
```

### 论证方法

#### 1. 道理论证

阐述理论依据、政策精神。

```
创新是引领发展的第一动力，这是由创新的本质属性决定的。创新能够提高生产效率，优化资源配置，推动产业升级，从而为经济发展注入持久动力。正如习近平总书记指出的，"抓创新就是抓发展，谋创新就是谋未来"。
```

#### 2. 事例论证

用具体案例支撑观点。

```
以深圳为例，这座改革开放的前沿城市，正是依靠创新驱动实现了从"制造"到"智造"的华丽转身。华为、腾讯、大疆等一批创新型企业在这里崛起，成为引领行业发展的排头兵。深圳的实践证明，创新是城市发展的不竭动力。
```

#### 3. 对比论证

正反对比增强说服力。

```
纵观历史，凡是重视创新的国家和民族，都能在竞争中占据优势；而固步自封、缺乏创新的，终将落后于时代。当年英国凭借工业革命成为"日不落帝国"，而清朝闭关锁国最终沦为半殖民地。历史的教训告诉我们，创新则兴，守旧则衰。
```

### 论据积累

| 领域 | 可用案例 |
|------|---------|
| 科技创新 | 北斗、嫦娥、天问、5G、量子计算 |
| 企业创新 | 华为、比亚迪、宁德时代 |
| 地方实践 | 深圳、杭州、合肥 |
| 历史典故 | 四大发明、洋务运动 |
| 国际对比 | 硅谷、以色列 |

## 四、结尾

### 结尾模式

#### 1. 总结式

```
综上所述，创新驱动发展不仅是时代的要求，更是历史的选择。我们要以更大的决心、更有力的措施，推动创新成为高质量发展的强大引擎，为实现中华民族伟大复兴的中国梦注入不竭动力。
```

#### 2. 展望式

```
站在新的历史起点上，创新驱动发展的号角已经吹响。让我们以"敢为天下先"的勇气和"功成不必在我"的境界，勇攀科技高峰，必将迎来创新发展的美好明天。
```

#### 3. 号召式

```
创新发展，时不我待；科技强国，任重道远。让我们凝心聚力、锐意进取，用创新的智慧点亮民族复兴的希望之光，用奋斗的汗水浇灌科技强国的梦想之花！
```

## 常见错误

::: warning ⚠️ 避免以下问题
1. **立意跑题**：没有紧扣材料主题
2. **论点空洞**：没有实质内容
3. **论据陈旧**：案例老套或不准确
4. **结构混乱**：逻辑不清晰
5. **首尾脱节**：开头结尾与主体不呼应
6. **字数不足**：没有达到要求字数
:::

## 配套练习

建议结合历年真题进行练习：
1. 先分析材料，确定立意
2. 列出分论点提纲
3. 限时完成全文
4. 对照范文修改提升
