# 📜 申论

申论是公务员考试的重要组成部分，主要考查阅读理解、综合分析、提出问题和文字表达能力。

## 考情概述

| 考试类型 | 时间 | 材料 | 题量 |
|---------|------|------|------|
| 国考（地市级） | 150分钟 | 5000-6000字 | 5题 |
| 国考（副省级） | 180分钟 | 6000-8000字 | 5题 |
| 省考 | 150分钟 | 5000-7000字 | 4-5题 |

## 五大题型

### 📝 概括归纳
- [概括归纳技巧](./gaigui.md)
- 单一要素概括 - 问题/原因/影响/对策
- 综合概括 - 多要素整合

### 🔍 综合分析
- [综合分析技巧](./fenxi.md)
- 词句理解 - 解释特定词句含义
- 观点评论 - 对观点进行评析
- 关系分析 - 分析多个事物关系

### 💡 提出对策
- [提出对策技巧](./duice.md)
- 直接对策 - 材料中有提示
- 间接对策 - 根据问题反推

### 📄 公文写作
- [公文写作技巧](./gongwen.md)
- 倡议书、建议书、讲话稿
- 调研报告、工作方案
- 新闻稿、短评

### ✍️ 大作文
- [大作文技巧](./zuowen.md)
- 立意确定 - 紧扣主题
- 结构布局 - 五段三分式
- 论证方法 - 举例、道理、对比

## 答题技巧

::: tip 💡 申论答题核心
1. **材料为王** - 答案都在材料中，忠于材料
2. **问什么答什么** - 审清题目要求
3. **条理清晰** - 分条作答，标序号
4. **语言规范** - 使用书面语，避免口语化
:::

## 常用结构

### 概括归纳结构
```
总括句 + 分点阐述
一、...
二、...
三、...
```

### 大作文结构（五段三分式）
```
第一段：开头（引出话题 + 提出论点）
第二段：分论点一 + 论证
第三段：分论点二 + 论证
第四段：分论点三 + 论证
第五段：结尾（总结升华）
```
