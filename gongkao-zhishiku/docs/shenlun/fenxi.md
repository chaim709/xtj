# 综合分析

综合分析考查对材料中观点、现象的理解和评析能力，是申论中较有难度的题型。

## 题型概述

| 特征 | 说明 |
|------|------|
| 分值 | 15-20分 |
| 字数 | 200-400字 |
| 难度 | ⭐⭐⭐⭐ |
| 核心 | 先解释，后分析，再升华 |

## 常见类型

| 类型 | 问法示例 |
|------|---------|
| 词句理解 | 请谈谈对"XXX"这句话的理解 |
| 观点评论 | 请对"XXX"这一观点进行评析 |
| 现象分析 | 请分析这一现象产生的原因和影响 |
| 关系分析 | 请分析A和B之间的关系 |

## 答题结构

### 词句理解类

```
1. 解释含义（是什么）
   ↓
2. 展开分析（为什么/怎么样）
   - 原因/背景
   - 问题/影响
   - 做法/经验
   ↓
3. 总结升华（怎么办/意义）
```

### 观点评论类

```
1. 亮明态度（支持/反对/辩证看待）
   ↓
2. 分析理由
   - 正面论证
   - 反面论证
   ↓
3. 得出结论
```

## 解题步骤

```
1. 审题
   ├── 确定分析对象（词句/观点/现象）
   ├── 明确字数要求
   └── 找到材料范围
       ↓
2. 定位材料
   ├── 找到关键词句出处
   └── 阅读上下文语境
       ↓
3. 提炼要点
   ├── 解释层面：直接含义
   ├── 分析层面：原因、影响、问题
   └── 升华层面：对策、意义
       ↓
4. 组织答案
   ├── 逻辑清晰
   └── 语言规范
```

## 例题解析

### 例1：词句理解

**题目：** 请谈谈你对"城市的温度"这句话的理解。（20分，不超过300字）

**材料线索：**
> 一座城市不仅需要高楼大厦，更需要有温度的公共服务...让弱势群体感受到社会的关怀...

**答案结构：**

```
这句话是指城市在追求经济发展的同时，更要注重人文关怀，让每一位市民感受到温暖和尊重。【解释含义】

具体表现在：
一、公共服务有温度。完善社会保障体系，关注弱势群体需求，提供贴心便民服务。
二、城市管理有温度。执法柔性化，保留城市烟火气，兼顾民生需求。
三、城市规划有温度。保护历史文化遗存，建设宜居社区，营造温馨生活环境。
【展开分析】

因此，城市建设应坚持以人为本，在追求发展速度的同时，更加注重发展的质量和温度，让城市成为人民美好生活的载体。【总结升华】
```

### 例2：观点评论

**题目：** 有人认为"发展经济必然牺牲环境"，请对这一观点进行评析。（15分，不超过250字）

**答案结构：**

```
这一观点过于片面，经济发展与环境保护并非完全对立。【亮明态度】

一方面，传统粗放型发展模式确实造成了环境污染，付出了生态代价。但另一方面，绿色发展理念的践行证明，经济发展可以与环境保护相协调。例如，绿色产业、循环经济的兴起，不仅保护了环境，也创造了新的经济增长点。【辩证分析】

因此，发展经济不必以牺牲环境为代价，关键在于转变发展方式，坚持走绿色发展道路，实现经济效益与生态效益的双赢。【得出结论】
```

## 核心技巧

### 1. 找准切入点

- 词句理解：先找到原文出处
- 观点评论：先判断观点性质
- 现象分析：先明确现象本质

### 2. 层层递进

```
表层含义 → 深层含义 → 延伸意义
现象描述 → 原因分析 → 对策建议
```

### 3. 结合材料

::: tip 💡 核心原则
- 分析内容必须有材料支撑
- 不能脱离材料空谈道理
- 材料中的案例可作为论据
:::

### 4. 升华有度

- 升华要与材料主题相关
- 避免空洞口号
- 落脚到具体对策或意义

## 常见错误

::: warning ⚠️ 避免以下问题
1. **不解释直接分析**：跳过含义解释
2. **脱离材料**：自己发挥过多
3. **观点不明**：态度模糊
4. **分析单一**：只有一个角度
5. **升华过度**：空洞无物
6. **结构混乱**：缺乏逻辑层次
:::
