# 📋 公共基础知识

公共基础知识（简称"公基"）是事业单位考试的重要科目，内容涵盖政治、法律、经济、管理、公文、科技等多个领域。

## 考情概述

| 考试类型 | 题量 | 时间 | 特点 |
|---------|------|------|------|
| 事业单位 | 80-120题 | 90-120分钟 | 知识面广、记忆为主 |
| 三支一扶 | 60-100题 | 90分钟 | 侧重基层实务 |

## 知识模块

### 一、马克思主义哲学

#### 唯物论

| 概念 | 内容 |
|------|------|
| 物质 | 客观实在性是物质的唯一特性 |
| 意识 | 物质的产物，人脑的机能 |
| 运动 | 物质的存在方式，绝对的 |
| 静止 | 运动的特殊状态，相对的 |

#### 辩证法

**三大规律：**

| 规律 | 内容 | 意义 |
|------|------|------|
| 对立统一 | 矛盾是事物发展的动力 | 核心规律 |
| 质量互变 | 量变引起质变 | 发展形式 |
| 否定之否定 | 事物发展是螺旋式上升 | 发展方向 |

**五对范畴：**
- 现象与本质
- 内容与形式
- 原因与结果
- 必然与偶然
- 可能与现实

#### 认识论

```
实践 → 认识 → 再实践 → 再认识（循环往复）
```

- 实践是认识的来源、动力、检验标准
- 认识具有反复性、无限性、上升性

### 三、法律知识

#### 法理学基础

| 概念 | 内容 |
|------|------|
| 法的特征 | 国家强制性、规范性、普遍性 |
| 法律关系 | 主体、客体、内容 |
| 法律责任 | 民事、行政、刑事责任 |

#### 宪法

**国家机构：**

| 机构 | 性质 | 职能 |
|------|------|------|
| 全国人大 | 最高权力机关 | 立法、监督 |
| 国务院 | 最高行政机关 | 行政管理 |
| 最高法院 | 最高审判机关 | 司法审判 |
| 最高检察院 | 最高检察机关 | 法律监督 |

#### 民法典

**总则编要点：**
- 民事主体：自然人、法人、非法人组织
- 民事权利：人身权、财产权
- 民事法律行为：有效、无效、可撤销

**物权编要点：**
- 所有权：占有、使用、收益、处分
- 用益物权：土地承包经营权、建设用地使用权
- 担保物权：抵押权、质权、留置权

#### 刑法

**犯罪构成四要件：**
1. 犯罪客体
2. 犯罪客观方面
3. 犯罪主体
4. 犯罪主观方面

**刑罚种类：**

| 类型 | 主刑 | 附加刑 |
|------|------|--------|
| 内容 | 管制、拘役、有期徒刑、无期徒刑、死刑 | 罚金、剥夺政治权利、没收财产、驱逐出境 |

#### 行政法

**行政行为：**
- 行政许可
- 行政处罚
- 行政强制
- 行政确认

**行政救济：**
- 行政复议：上级行政机关
- 行政诉讼：人民法院

### 四、经济知识

#### 市场经济

| 概念 | 内容 |
|------|------|
| 市场机制 | 价格机制、供求机制、竞争机制 |
| 市场结构 | 完全竞争、垄断竞争、寡头垄断、完全垄断 |
| 市场失灵 | 外部性、公共产品、信息不对称 |

#### 宏观调控

**财政政策：**
- 扩张性：增加支出、减少税收
- 紧缩性：减少支出、增加税收

**货币政策：**
- 扩张性：降低利率、增加货币供应
- 紧缩性：提高利率、减少货币供应

4. **时政关注**：结合时事热点
:::
