# 泗洪单招知识库

## 简介

本知识库用于管理泗洪县事业单位单招考试资料。

## 目录结构

- `00-索引/` - 索引和统计信息
- `01-真题库/` - 历年真题和模拟题
- `02-知识点库/` - 按科目分类的知识点
- `03-冲刺资料/` - 冲刺阶段重点资料
- `04-押题预测/` - 考前押题和预测
- `05-原始文件/` - 原始 PDF/Word 文件

## 考试科目

### 公共基础知识
- 法律
- 政治
- 经济
- 文化

### 职业能力倾向测验
- 言语理解
- 数量关系
- 判断推理
- 资料分析

## 使用说明

1. 原始文件放在 `05-原始文件/`
2. 使用 document-parser 处理后生成 Markdown 笔记
3. 使用 knowledge-extractor 提取知识点
4. 使用 question-indexer 生成题目索引
