哲学3部分主要涉及的是关于认识论和唯物论的问题。
一、单选题
1.实践的观点是马克思主义的首要的基本观点，是理解马克思主义哲学的基础。下列关
于实践的说法，不正确的是（ ）。
A.实践是主观见之于客观的东西
B.实践具有社会历史性
C.实践是社会生活的本质
D.实践是最根本的认识方法
【解析】1.选非题。本题涉及实践基础理论的考点。实践的观点是马克思主义的首要的
基本观点（单选题考点，常考查）。A项正确：实践是人类能动地改造世界的客观物质活动，
强调主体是人，人必须发挥能动性，体现主观方面。改造的对象是客观物质活动，是客观方
面。比如在一大片荒地上设计一栋建筑，需发挥能动性，考虑建成什么形状、用什么材料、
采用何种结构更好等，属于能动方面，通过实践将想法作用在客观物质上，建成想象中的建
筑，这就是主观见之于客观。B项正确：强调实践不能脱离社会，受历史条件的制约。C项
正确：“实践是社会生活的本质”是马克思所说，是正确的。整个社会生活存在和发展有一
定的基础，首先有物质性东西，要先吃饱，吃的食物是通过实践而来的，因此实践是人类存
在和发展的基础，通过生产实践可获得基础物质资料，这种情况下实践是社会生活的本质。
D项错误：认识最想要达到对事物本质的认识，虽通过实践可以获得认识，但获得的这个认
识不一定都是对本质的认识，因此最根本的认识方法是矛盾分析法（由唯物辩证法的地位决
定，对立统一规律是唯物辩证法的实质和核心）。【选D】
【注意】
1.需掌握A项的表述是正确的，实践是主观见之于客观的东西，是沟通主观和客观的桥
梁。主观见之于客观指将主观加到客观上，比如盖楼过程中，主观想象一个楼的形状，可通
过实践变为一栋楼，这个楼是客观的，存在之前的主观想法。
2.最根本的认识方法是矛盾分析法（认识到本质）。

【梳理】
1.实践的含义：实践是人类能动地改造世界的客观物质活动。改造的主体：人；“能动”
指人需发挥能动性；改造的对象：客观的物质。比如王某扛着锄头改造一大片荒地，属于实
践，此时需发挥其能动性（考虑这块地种什么收成更高、何时去耕作以及如何去耕作）。
2.实践的特点：
（1）客观物质性：强调构成实践的要素都是客观的。比如王某扛着锄头去耕地，此时
王某这个人（主体）、扛着的锄头（手段）、改造的这块土地都是客观的。
（2）主观能动性：人是有目的、有意识的去做，主要与动物的本能作区分。比如想要
有一个鸟窝一样的建筑，可以在大脑进行想象。
（3）社会历史性：可拆为两部分。
