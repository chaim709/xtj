# 公考知识库使用指南

## 📚 知识库概述

这是一个基于 Obsidian 的公考资料知识库，用于管理和利用各类公考学习资料。

## 📁 目录结构

- **00-索引/** - 知识库索引和使用指南
- **01-真题库/** - 历年真题及解析
  - 国考/
  - 省考/
  - 事业单位/
- **02-知识点库/** - 按科目和模块整理的知识点
  - 行测/（言语理解、数量关系、判断推理、资料分析、常识判断）
  - 申论/
- **03-机构资料/** - 各培训机构的教学资料
- **04-名师讲义/** - 名师讲义和笔记
- **05-时政热点/** - 时政新闻和热点分析
- **06-学员资料/** - 学员笔记、错题集、学习计划
- **07-原始文件/** - 原始文件存储（PDF、视频、文档、图片）
- **08-模板/** - 笔记模板

## 📝 命名规范

### 真题文件
格式：`[年份]-[考试类型]-[科目]-[题型].md`
示例：`2024-国考-行测-言语理解.md`

### 知识点文件
格式：`[科目]-[模块]-[知识点].md`
示例：`行测-数量关系-排列���合.md`

### 机构资料
格式：`[机构名]-[老师名]-[科目]-[主题].md`
示例：`机构A-张老师-行测-逻辑填空技巧.md`

## 🔍 快速查找

### 使用 Obsidian 搜索
```
#真题 #排列组合 #国考
```

### 使用 Dataview 查询
```dataview
TABLE 年份, 科目, 题型
FROM #真题
WHERE 年份 >= 2020
SORT 年份 DESC
```

## 🚀 快速开始

1. 将原始资料放入 `07-原始文件/` 对应目录
2. 使用批量处理工具提取内容
3. AI 生成结构化笔记
4. 添加标签和链接
5. 开始使用和检索
