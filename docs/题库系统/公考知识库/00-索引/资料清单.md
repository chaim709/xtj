# 资料清单

## 统计信息

```dataview
TABLE count(rows) as 数量
FROM ""
GROUP BY file.folder
SORT 数量 DESC
```

## 真题统计

```dataview
TABLE 年份, 科目, count(rows) as 题目数
FROM #真题
GROUP BY 年份, 科目
SORT 年份 DESC
```

## 知识点统计

```dataview
TABLE count(rows) as 数量
FROM #知识点
GROUP BY 科目
```

## 最近更新

```dataview
TABLE file.mtime as 更新时间
FROM ""
SORT file.mtime DESC
LIMIT 20
```
