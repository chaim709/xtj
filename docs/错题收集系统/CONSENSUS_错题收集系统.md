# 线下机构学员错题收集系统 - 共识文档

**确认时间**: 2026-01-29

---

## 确认的需求

### 核心功能
1. **题目管理** - 导入、清洗、分类、编辑
2. **题册管理** - 创建题册、组织题目
3. **题册生成** - 生成带二维码的PDF/Word（只有题目，无答案）
4. **扫码录题** - H5页面，学员扫码勾选错题，提交后查看解析
5. **错题收集** - 按学员/题册/知识点统计错题

### 关键决策
| 问题 | 决策 |
|------|------|
| 题册内容 | 只显示题目，答案扫码后查看 |
| 现有数据 | 删除，重新开始 |
| 部署方式 | 先本地测试，后续部署公网 |
| 学员识别 | 姓名 + 手机号 |

### 技术方案
- 后端: Flask + SQLAlchemy
- 数据库: SQLite
- PDF生成: WeasyPrint
- Word生成: python-docx
- 二维码: qrcode

---

## 验收标准

1. ✅ 能导入题目并分类管理
2. ✅ 能创建题册并组织题目
3. ✅ 能生成带二维码的PDF/Word
4. ✅ 学员扫码能勾选错题并提交
5. ✅ 提交后显示错题解析
6. ✅ 管理员能查看学员错题数据
